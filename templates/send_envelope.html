How It Works

The API call uses POST /envelopes with templateId and templateRoles to route to the right signer role.

Tab values map to named fields inside the template document — no document editing required.

POST /restapi/v2.1/accounts/{id}/envelopes { "templateId": "...", "status": "sent", "templateRoles": [{ "email": "signer@co.com", "name": "Jane Smith", "roleName": "Signer", "tabs": { "textTabs": [{ "tabLabel": "Company", "value": "Acme Corp" }] } }] }
