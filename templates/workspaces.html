Dynamic hub pattern
  1. ERP / procurement system triggers new vendor onboarding
  2. POST /workspaces creates hub for {{ demo.vendor_name }}
  3. eSignature API attaches draft MSA envelope to workspace
  4. API adds upload requests (SOC 2, insurance, STD 204)
